 Image Processing Project
 Overview

This project is a comprehensive implementation of fundamental and advanced Image Processing techniques using a unified GUI-based application.

It covers a wide range of operations including:

Point Operations
Color Image Processing
Histogram Processing
Filtering Techniques
Image Restoration
Image Segmentation
Edge Detection
Mathematical Morphology

The system allows users to apply different operations and visualize both input and output images side-by-side.
