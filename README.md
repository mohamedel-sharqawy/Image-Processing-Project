#  Image Processing Toolkit with GUI



## 📌 Overview
This project is a complete **Image Processing Toolkit** that implements a wide range of image processing techniques using a **GUI application**.

It allows users to apply operations and compare results using **input vs output visualization**.



## 🎯 Key Features

### 🔹 Point Operations
- Addition  
- Subtraction  
- Division  
- Complement  

### 🔹 Color Image Processing
- Adjust Red Channel  
- Swap Channels (Red → Green)  
- Remove Red Channel  

### 🔹 Histogram Processing
- Histogram Stretching  
- Histogram Equalization  

### 🔹 Neighborhood Processing
#### Linear Filters
- Average Filter  
- Laplacian Filter  

#### Non-Linear Filters
- Maximum  
- Minimum  
- Median  
- Mode  

### 🔹 Image Restoration
#### Salt & Pepper Noise
- Average Filter  
- Median Filter  
- Outlier Method  

#### Gaussian Noise
- Image Averaging  
- Average Filter  

### 🔹 Image Segmentation
- Global Thresholding  
- Automatic Thresholding  
- Adaptive Thresholding  

### 🔹 Edge Detection
- Sobel Operator  

### 🔹 Mathematical Morphology
- Dilation  
- Erosion  
- Opening  
- Boundary Extraction  



## 🖥️ GUI Features
- Simple interface  
- Upload images  
- Apply operations  
- Show results using subplots  



## 🛠️ Technologies Used
- Python  
- OpenCV  
- NumPy  
- Matplotlib
